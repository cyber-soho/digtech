### How to Run

**Python files** — Requires Python 3.8+. Some scripts need extra packages:
```bash
pip install scikit-learn pandas numpy    # For examples 1.1, 1.3, 1.5
pip install python-pptx                  # For the PPTX builder only
```

**HTML files** — Open directly in any modern browser (Chrome, Firefox, Safari, Edge). No server or dependencies needed.
